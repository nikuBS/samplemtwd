baro Box 임대

내용

baro Box란?

전세계 주요국에서 무선인터넷을 이용하실 수 있도록 T로밍 전용 와이파이 기기를 임대해드리는 서비스입니다.

baro Box 예약 방법

온라인 또는 고객센터를 통한 사전예약 후 임대하실 수 있습니다.(출국 2일 전 예약 가능)
  • 온라인 사전 예약
    • baro Box 임대 > baro Box 예약 메뉴를 통해 사전 예약하실 수 있습니다.
  • 고객센터 사전 예약
    • T로밍 고객센터(02-6343-9000)를 통해서도 사전 예약하실 수 있습니다. (SK텔레콤 휴대폰에서 통화료 무료)
  • 로밍센터에서 기기 수령 시 신분증 확인 후 수령하실 수 있습니다. 기기 수령 후 로밍센터에서 요금제 가입을 도와드립니다.
  • 대구 지역은 온라인 또는 고객센터 사전 예약 후 SK텔레콤 황금점 매장에서도 수령하실 수 있습니다.(주소: 대구 수성구 동대구로 98, 일요일 휴무)

baro Box 이용 방법(임대)

  • baro Box 예약
  • 공항 로밍센터에서 기기 수령 및 여행 국가에 따라 baro Box 상품 가입
  • baro Box 전용 요금제로 LTE 및 3G 데이터 사용
