모바일 T world 권한 필요 안내

내용

모바일 T world 사용을 위해선
다음과 같은 권한이 필요합니다.

필수 접근권한

  • 전화 권한
    로그인 하실 때 접속하신 기기의 전화번호 확인 및 회원가입 하실 때 구글 계정상태 등의 확인과 공인인증서 사용, 매장찾기 등에서 직접 전화 연결 시 필요합니다.
  • SMS 권한
    업무처리 과정에서 SMS인증 및 단말기 분실 시 로그인 접속 해제, T알림(PUSH)등 문자 메시지 수신을 위해 사용됩니다.

필수 권한을 허용 해 주셔야 모바일T world를 이용하실 수 있습니다.

선택 접근권한

  • 주소록 권한
    회원 가입 간소화를 위한 구글 계정 정보 사용과 무료 문자 보내기, 데이터 선물하기 주소록 연동 시 연락처 확인 하기 위해 사용됩니다.
  • 사진/미디어/파일 권한
    무료 문자 보내기, 사진 첨부, 공인 인증서 가져오기/내보내기 기능을 위해 사용됩니다.
  • 위치 권한
    매장 및 AS 센터 찾기, 위치기반 PUSH 발송 등에서 기기 위치를 확인하기 위해 사용됩니다.
  • 카메라 권한
    무료 문자 보내기에서 카메라를 이용하여 사진을 찍고 전송을 위해 사용됩니다.

선택 접근권한은 동의하지 않아도 서비스 이용 가능하나, 해당권한이 필요한 기능은 제공이 제한될 수 있습니다.
