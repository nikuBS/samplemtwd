유심 잠금해제 주요용어

기기 소유권자

기기 주인을 말합니다. 신규가입이나 기기변경을 할 때, 대리점에서 고객 이름과 기기 정보를 전산에 입력하는데, 이때 등록된 고객이 해당 기기의 소유권자입니다. 이렇게 서비스 명의자와 기기 소유권자가 동일하기 때문에 전화번호 1개 당 1개의 기기에 소유권이 생기며, 서비스를 해지하거나 기기변경 후 남은 공기기에는 소유권자가 존재하지 않습니다. 고객님의 기기에 친구가 유심을 끼워 사용하는 경우 기기 소유권자는 고객님으로 유지됩니다.

유심기변 기기

고객님이 유심기변 하여 사용중인 기기를 의미합니다. 고객님의 기기에 있는 유심을 친구의 기기에 꽂아서 이용하신다면 친구의 기기가 고객님의 유심기변 기기가 되는 것입니다.

원 소유 기기

SK텔레콤에 고객님의 이름으로 등록된 기기를 의미합니다. 신규 가입고객, 전산기변(확정기변 포함) 고객, 명의변경 고객이 소지한 기기가 원 소유 기기이고, 고객님은 그 기기의 기기 소유권자가 되시는 겁니다.

현 사용 기기

현재 고객님이 사용 중인 기기로, 원 소유 기기 또는 유심기변 기기에 상관없이 현재 고객님이 유심 카드를 꽂아서 사용하면 모두 현 사용 기기입니다.

공기기

개통 이력이 없는 신규 기기거나 기기변경, 해지 등으로 회선과 분리된 기기를 의미하며 공기기는 소유권자가 따로 없습니다.
