휴대폰 다른 이동통신사 이용 신청

기기를 다른 이동통신사에서도 사용할 수 있도록 이동통신회사 간 정보 공유를 요청하는 것으로, 사용하려는 기기의 원래 이동통신회사에 '휴대폰 타사 이용 신청'을 하셔야 합니다.

휴대폰 보호 서비스

타인이 내 기기를 이용하지 못하도록 차단하는 무료 부가서비스로 지점을 방문하시거나 고객센터, T월드에서 가입하실 수 있습니다. 고객님의 기기에 고객님의 유심을 장착한 경우에만 서비스 가입이 되고 고객님의 기기에는 고객님의 유심만 작동합니다. 유심 기변을 하여 현 사용 기기인 경우 보호 대상이 아니며 대리점에서 기기변경이나 번호 변경을 한 경우에는 서비스가 유지됩니다.
