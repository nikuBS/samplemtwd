미성년자 가입

미성년 자녀의 이용요금을 알려드리며, 팅 요금제 명의자를 대상으로 고객이 희망할 경우 무선인터넷 접속을 차단해 주는 무료 부가서비스를 안내해 드립니다.
안내 유형을 선택하세요.
  • 자녀요금 통보서비스 안내

    미성년 자녀의 이용요금을 법정대리인(부모)에게 문자로 알려드리는 서비스입니다.

    서비스 신청 대상
    20세 미만 미성년 고객
    통보 대상 이용요금
    미성년자 명의 회선의 국내통화료, Data 사용료, Data 정보이용료, 소액결재(단, 팅 요금제의 경우 한도 관리되는 요금 항목은 통보 대상에서 제외)
    팅 요금제
    Data 정보료 + 기본제공 통화 소진 후 추가로 발생되는 요금
    일반요금제
    국내통화료 + Data 사용료 + Data 정보이용료 + 소액결제

    * 통보 서비스 항목의 합산 요금이 5천원/1만원/2만원/4만원/6만원/8만원/10만원/15만원 초과할 경우 각각 문자 통보
