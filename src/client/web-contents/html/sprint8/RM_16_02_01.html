  • LTE 자동로밍

    해외에서도 3G 자동로밍보다 훨씬 빠른 데이터로밍 서비스를 이용하실 수 있으며, 음성 및 SMS는 3G 자동로밍과 동일하게 이용하실 수 있습니다.

    현지 이동통신사업자의 통신 사정에 의해 LTE 를 이용할 수 없는 경우 3G로 연결됩니다.

    휴대폰에서 LTE로밍 설정

    LTE 데이터로밍을 이용하기 위해 휴대폰에서 LTE 사용모드 및 지정된 사업자 망을 설정해 주세요. 현지 사정에 따라 LTE 데이터 통신이 원활하지 않은 경우 3G 데이터로 접속될 수 있습니다.

    • 스마트폰(iOS : 아이폰 6S iOS9 기준)
      • 1´설정´에서 ´셀룰러´ 선택
      • 2셀룰러 데이터 ON
      • 3셀룰러 데이터 옵션 > 데이터로밍 ON
      • 4셀룰러 데이터 옵션 > LTE 활성화
        (음성 및 데이터 / 데이터 전용 중 선택)
    • 스마트폰(안드로이드)
      • 1환경 설정에서 ´T로밍´ 선택
      • 2´데이터로밍 설정´, ´LTE로밍 사용´ 체크

    로밍 음성/영상/문자 이용 시 국가별 별도 요금이 발생됩니다.
    문의: T로밍 고객센터 02-6343-9000
    (SK텔레콤 폰에서 무료, 24시간 연중무휴)

    LTE로밍 수동 설정

    • 일반폰

      메뉴 > 1.T서비스 > 1.글로벌로밍 > 1.시스템 선택 > 확인 > 2.수동선택 > 국가별 할인망 선택

    • 스마트폰(iOS)
      • 1´설정´에서 ´네트워크 사업자´ 선택
      • 2자동 OFF
      • 3제휴 이통사망 선택
    • 스마트폰(안드로이드)
      • 1´설정´에서 ´T로밍´ 선택
      • 2로밍 이동 통신사 선택
      • 3수동 선택
      • 4해외 할인망 선택

      대부분의 국가에서 SK텔레콤 지정 이동통신사망으로 자동 설정이지만, 혹시라도 안될 경우에는, 위 내용을 참고하셔서 수동설정해 주세요. 지정 이동통신사망 이외에서 로밍 이용 시 과다한 요금이 발생할 수 있습니다.

    자동로밍 서비스란 별도 신청 없이 내 휴대폰, 내 번호 그대로 해외에서 바로 사용 가능한 서비스입니다.
