대표 회선에 로밍 데이터 300MB 제공

baro Box 이용 가능 거리에서 벗어났을 때 활용하세요.
(데이터 로밍 사용 설정 필요)

  • 혜택 대상
    baro Box를 대여할 때 등록한 대표 회선(본인 명의)에 LTE/3G 데이터 제공(SK텔레콤 고객님에게만 제공되는 혜택입니다.)
  • 혜택 대상 요금제
    baro Box 일본, 미국, 캐나다 / baro Box 아시아 / baro Box 글로벌
  • 추가 제공 데이터 300MB를 모두 쓰신 경우에는 ‘데이터로밍무조건차단’에 자동으로 가입됩니다. 단, 데이터 로밍이 차단되어도, baro Box는 정상적으로 이용하실 수 있습니다.
  • baro Box 요금제를 이용할 때 다른 로밍 요금제에 가입되어 있다면, 추가 제공 데이터를 받으실 수 없습니다.
