모바일 T 안심인증번호 확인 방법

내용

SMS가 수신 안 되신 경우

  1. 모바일 T 안심인증 SMS는 114 번호로 발신되오니 해당 번호의 단말기 스팸설정 여부 및 스팸편지함을 확인해 주세요.
  2. 스팸필터링 부가서비스 가입 후 114 번호를 스팸설정 하신 경우 스팸설정을 해제해 주세요.
  3. 착신 관련 서비스에 가입되어 있는 경우 SMS가 수신되지 않을 수 있습니다.
  4. 현재 사용중인 휴대폰이 USIM 단독상태이거나 휴대폰과 USIM카드가 분리된 상태에서는 SMS를 받으실 수 없습니다.

SMS가 수신 되신 경우

  1. 휴대폰의 데이터 네트워크 접속상태를 확인해 주세요.
  2. 고객님의 휴대폰에 최신버전(14.06.24 기준/버전 4.03이상)의 모바일 T world App. 설치되어 있는지 확인해 주세요.
