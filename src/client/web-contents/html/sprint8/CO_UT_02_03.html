내용

모바일 T world 사용을 위해선
다음과 같은 권한이 필요합니다.

  • 연락처 권한
    무료 문자 보내기, 데이터 선물하기 주소록 연동 시 연락처 확인을 위해 사용됩니다.
  • 사진 권한
    무료 문자 보내기에서 사진 첨부 시 사용하기 위해 사용됩니다.
  • 위치 권한
    매장 및 AS 센터 찾기 등에서 현재 위치를 확인하여 지동사에 위치를 표기하기 위해 사용됩니다.
  • 카메라 권한
    무료 문자 보내기에서 카메라를 이용하여 사진을 찍고 전송을 위해 사용됩니다.

위 접근권한은 동의하지 않아도 서비스 이용이 가능하나, 해당 권한이 필요한 기능은 제공이 제한될 수 있습니다.
