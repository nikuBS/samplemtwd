  • 개발팀 확인 요청사항

    레이어 팝업 닫았을 때 포커스가 처음 클릭 했던 버튼 으로 이동해야 하는 이슈 는 개발팀에서 확인 부탁드립니다.
    해당 레이업 팝업 클릭 이벤트를 수정해야 하는 부분이라서 제가 함부로 수정을 할 수가 없어서 이부분은 skip 하였습니다.

    aria-pressed="true / false" 와 aria-selected="true /false " 을 적용해야 하는 탭메뉴, 아코디언 메뉴 들
    기존 버튼 클릭시 .active 클래스명도 함께 토글 이 되어야 되기 때문에 제가 추가한 aria 관련 스크립트가 기존 클릭 이벤트와 꼬이지 않게 적용 부탁 드립니다.

    달력 을 웹접근성에 맞게 수정하기 위해서 현재 운영에서 사용중인 datepicker.js를 수정을 하였습니다.
