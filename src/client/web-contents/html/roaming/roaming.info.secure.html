  • 매일 3분까지는 무료 통화가 제공됩니다.

"비행기가 연착돼서

  • 음성통화는 초당 계산되어 쓴 만큼만 합리적으로 청구됩니다.

자동안심 T로밍 Q&A

Q 자동안심 T로밍을 이용하려면 어떻게 해야 하나요? 가입 신청을 해야 하나요?

자동안심 T로밍을 이용하기 위해 별도의 가입 신청은 필요 없습니다. 로밍 요금제에 가입하지 않아도 해외에서 로밍 서비스를 이용하시면 ‘자동안심 T로밍’ 서비스가 자동으로 적용됩니다.

  • 음성통화: 1일 기준 이용 시간에 따라 자동으로 30분 최대 10,000원 서비스 적용
  • 데이터 안심 서비스: 1일 이용료가 5,000원(부가세 포함)을 초과할 때 자동으로 데이터 안심 서비스 가입. 이후 24시간 동안 속도 제한 상태에서 무제한 사용(단, 제휴국이 아닌 경우 5,000원 초과 시 데이터 자동 차단)
Q 통화 시간이 30분이 넘을 경우, 요금은 어떻게 청구되나요?

1일 총 음성통화 시간이 30분을 넘지 않을 경우 10,000원만 청구됩니다. 단, 30분을 초과할 경우 초과한 시간(초)만큼 해당 국가의 초당 요금으로 계산되어 청구됩니다.

예) 홍콩에서 홍콩으로 30분 10초간 통화 시 총 10,064원 청구

  • 30분 통화: 10,000원
  • 30분 초과 통화: 10초 x 6.4원(현지 통화 시 초당 통화 요금) = 64원

-> 총 10,064원

Q 하루의 기준은 어떻게 되나요?

한국 시간 기준으로 00시~24시 입니다.

Q 속도 제한 이후 더 빠른 속도로 데이터를 사용하려면 어떻게 해야 하나요?

속도가 제한된 이후 더 빠른 속도로 데이터를 사용 하시려면 baro 3/4/7GB, baro OnePass 300/500/VIP/Data VIP(기본형, 기간형) 등의 로밍 요금제를 신청해 보세요. 해외에서도 가입 후 바로 이용하실 수 있습니다.
