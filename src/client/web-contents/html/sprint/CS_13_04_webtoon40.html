콘텐츠 이용사이트 회원가입을 통한 유료결제 등록일 2020.08.06

오~ 회원가입하면 다운로드 무료로 이용가능하다고? 가입해야지
휴대폰 본인 인증을 거쳐서~ 회원가입완료~ 응 ??? 요금청구에 해당사이트 회원가입 시 결제 인증이 이루어져 25,000원이 휴대폰 소액결제????
회원가입을 통해 유료결제 되는 피해를 입었구나??? 회원가입 시 약관이나 계약조건 등을 확인하고, 유료 회원가입 여부, 자동 연장결제 여부 등 체크 해야 한다고!!
주민번호 휴대폰 번호 인증번호 등은 결제에 이용될 수 있으므로 불필요한 정보를 요구하는 사이트는 가급적 이용하지 않도록 함
결제 확인 알림 문자 수신여부 확인 및 확인 시 해당 결제대행회사 고객센터로 결제 취소를 요청해야해!
결제대행사? 찾아야겠어!
소액이라 무심코 지나치기 쉬우므로 매월 발송되는 요금청구서를 꼼꼼히 확인하여 본인이 모르는 결제내역이 있는지 확인할 필요가 있다고!
그렇구나!
콘텐츠 제공업체와 연락이 잘 안된다고 포기하지말고, 가입한 통신사업자(이동전화114)와 결제 대행회사에 먼저 이의제기 해야 해!
그래도 해결이 안된다면 소액결제중재센터(http://www.spayment.org/)에 도움을 요청하는거야!
