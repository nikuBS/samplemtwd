메신저 피싱 예방편 등록일 2019.11.21

휴대전화 메시지 대화내용
인성 : 친구야 미안한데 부탁이 있어.
현명 : 응. 갑자기 무슨 일 있어?
인성 : 지금 갑자기 은행 송금이 안돼서 그런데 OO은행에 50만원만 대신 보내 줄 수 있어?
현명 : 많이 급한 일인거야?
인성 : 응. 돈은 1시간 후에 바로 입금할테니 부탁 좀 할게.
현명 : 응 알았어.
현명이 있는 공간에 문을 열고 들어오는 인성
인성 : 아~ 진짜 덥다 더운데 뭐하고 있어? 시원한 곳 가자~
현명 : 어!??? 뭐야???? 방금 네가 보낸 메시지 확인하고 있었지! 갑자기 급한 일이라고 돈을 대신 입금해 달라고 하길래 무슨 일이 생긴 줄 알고 걱정 했잖아!
인성 : 내가? 급하게 돈을 입금해 달라고 했다고? 난 그런적 없는데~! 귀신이야? (소오름)
현명 : 그럼 나한테 메신저로 얘기한 사람이 네가 아니라는 거야? 아! 맞다 그러고 보니 요새 메신저 피싱 사기가 유행한다더니 (헉!)
인성 : 메신저 피싱? 그게 뭐야?
현명 : 메신저 피싱은 개인정보 유출 등으로 얻은 개인 아이디와 비밀번호를 통해 직장 동료나 친구, 가족처럼 메신저로 접근해서 돈을 요구하는 사기 수법이래.
인성 : 그럼 내 개인정보가 유출되었다는거네!
현명 : 휴우… 하마터면 나도 속아 넘어 갈뻔 했짢아.
메신저 피싱 방지 5계명
1. 메신저로 금전을 요구하는 경우 반드시 전화를 통해 확인하다.
2. 메신저를 통해 내 개인정보를 알려주지 않는다.
3. 메신저의 비밀번호를 정기적으로 변경하고 관리한다.
4. 공공장소에서는 메신저 사용을 자제한다.
5. 메신저 자체 보안 설정 및 보안 프로그램을 최신버전으로 유지한다.
현명 : 다시 속아 넘어가지 않으려면 메신저 피싱 예방법에 대해 같이 알아보자!
인성 : 응 이번에 다행이 돈을 송금하기 전에 사기라는걸 알아차려서 다행인데, 만약에 돈을 송금했다거나 사기를 당한 이후에는 어떻게 해야하지?
현명 : 그래 사기를 예방하는 것도 중요하지만 사기를 당한 이후의 신속한 행동도 아주 중요하지.
현명 : 메신저 피싱 사기를 당했을 때는 이렇게 행동하면 돼!
1. 메신저피싱 신고방법
2. 고객센터에 해킹ID신고
3. 은행에 지급정지 요청
4. 계좌이체내역서를 지참하여 경찰서에 사건신고접수
5. 경찰서에서 사실확인원을 발급받아 은행에 제출
인성 : 그렇구나 오늘처럼 돈을 요구하는 메시지가 오면 반드시 상대방에게 먼저 전화를 걸어서 확인을 해야 피해를 예방할 수 있어!
현명 : 응 맞아~ 그리고 가장 좋은 예방법은 개인정보가 유출되지 않도록 포털사이트랑 SNS의 비밀번호를 주기적으로 변경하는 것도 중요해!
현명 : 또 피해를 당한 경우 경찰서 (신고전화 112) 혹은 금융회사 콜센터를 통해 신속한 지급정지 요청을 하는 것도 잊으면 안돼!
