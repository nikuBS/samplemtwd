  • 해외 할인 이동통신망 이용상태 확인 방법
    대부분의 LTE·3G 휴대폰은 로밍하면 오토다이얼링 대기화면에 자동으로 이동통신사업자명이 표시됩니다.
    • 로밍 불가 요금제 사용 고객은 로밍 가능 요금제로 변경 후 이용하시기 바랍니다.
      (로밍 불가 요금제 : Ting/iNee 한도 가입자 및 PPS/HDN 선불 가입자 등)
    • 2G 단말 이용 고객은 요금제/부가 서비스에 가입하더라도 사용이 불가합니다.
      3G 단말 임대 후 이용하시기 바랍니다. (1회선만 이용 가능)
    • 서비스 가입 후 데이터 로밍을 1회도 사용하지 않은 경우에는 기본요금이 청구되지 않습니다.
    • 데이터 로밍을 1회 이상 사용하신 경우에는 신청 기간 기본요금 전액이 청구되며, 기본 제공 데이터 혜택 잔여분에 대해 이월 및 환불이 불가합니다.
    • 데이터 기본 혜택에는 LTE/3G 데이터 로밍 통화료만 적용됩니다. (데이터 정보이용료 미포함)
    • 해당 요금제 이용 시 음성 통화 매일 3분 무료 이용이 가능합니다.
    • 현지 사업자의 통신 사정에 의해 LTE 이용이 불가한 경우 3G로 연결됩니다.
    • 기간 내 일 300MB 데이터 용량 소진 시 속도제어가 적용됩니다. (400kbps 이하)
    • 데이터 차단을 원하실 경우에는 고객센터(T.114)로 문의 바랍니다.
    • 한국 귀국 후에는 네트웍을 자동 설정으로 재 변경해야 휴대폰을 이용할 수 있습니다.
