baro란?

이용이 불편하고 데이터 사용이 부담됐던 로밍을 바로잡은, SK텔레콤의 새로운 로밍 브랜드입니다.

01 baro 통화

SK텔레콤 고객님은 '기간'과 '데이터'만 선택하면 T전화로 baro 무료통화

baro 통화란?

해외에서 T전화를 이용해 고품질의 데이터 음성통화를 할 수 있는 서비스입니다. SK텔레콤 고객님이라면, baro 요금제에 가입하고 내 번호 그대로 통화할 수 있어요.

02 baro 요금제

baro 요금제는 글로벌 통합 로밍 요금제입니다. '기간'과 '데이터'만 결정하면 요금제 선택 완료.
나라 상관 없이 해외 나갈 땐 baro 요금제 하나면 끝!

    • 만 18세~29세 이하 고객님은 데이터 1GB를 추가로 받을 수 있는 baro YT 4GB 요금제에 가입하세요.
    • 만 18세~29세 이하 고객님은 데이터 1GB를 추가로 받을 수 있는 baro YT 5/8GB 요금제에 가입하세요.
    • 기간형 상품은 한국 시간 기준 고객 지정 시각부터 24시간 이용 가능합니다.
    • 데이터 무제한, 음성 30분, 문자 30건의 혜택을 누려보세요.
03 baro Box

여러 명이 함께 여행간다면? 해외에서 데이터를 나눠 쓰는 baro Box!

baro Box란?

전 세계 주요국에서 무선인터넷을 사용 할 수 있도록 T로밍 전용 Wi-Fi 기기를 임대해드리는 서비스입니다.
