테스트 환경

  • android 4.x
  • ios 최신
  • PC 크롬 최신

기본

  • HTML 문법오류 X
  • 웹접근성 고려 ARIA 사용
  • 기본 코딩가이드 (동일폴더 파일참고)

SCSS - font-weight

  • font-weight:$fw-normal; // 300
  • font-weight:$fw-regular; // 400
  • font-weight:$fw-medium; // 500
  • font-weight:$fw-bold; // 700
  • font-weight:$fw-bolder; // 900

SCSS - font-family

  • font-family:$ff-number // "Helvetica Neue", Roboto, sans-serif;
  • font-family:$ff-tgothic-light // "TgothicLight", Roboto, sans-serif;
  • font-family:$ff-tgothic-medium // "TgothicMedium", Roboto, sans-serif;
  • font-family:$ff-tgothic-black // "TgothicBlack", Roboto, sans-serif;

SCSS - 모든 px 수치는 rem 통해 정의 (0, 퍼센테이지 제외)

  • width:rem(10px)
  • top:rem(10px)
  • margin:rem(10px)
  • padding-left:rem(10px)

SCSS - color (variabled.scss 참조)

  • color:$gray1
  • background:$gray1

SCSS - font-size (variabled.scss 참조)

  • font-size:$fs-12
