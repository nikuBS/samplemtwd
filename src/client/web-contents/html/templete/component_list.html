widget, components에 의존성을 띄는 경우 해당룰은 그대로 가져갑니다.

- .widget 다음에는 .widget-box 만 올수있음
- .widget-box 에는 정해진 class명 외에 사용금지 widgets.js 참조
- 특정 style이 필요할시 .widget .widget-box 두 태그의 구조만 바꾸지 않고 자유롭게 수정가능
ex) .style이 상위에 class지정
ex) .widget-box 내부에 태그추가 & class 추가

- .component 다음에는 .component-box 만 올수있음
- .component-box 에는 정해진 class명 외에 사용금지 widgets.js 참조
- 특정 style이 필요할시 .component .component-box 두 태그의 구조만 바꾸지 않고 자유롭게 수정가능
ex) .component 상위에 class지정
ex) .component-box 내부에 태그추가 & class 추가
