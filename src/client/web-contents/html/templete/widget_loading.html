skt_landing.action.loading

- .wrap 안에 컨텐츠만 대상으로 할수있음
- 대상의 outerWidth, outerHeight, offset().top, offset().left 를 계산하여 위치시킴
- .tw-loading의 갯수를 확인하여 현재 로딩중인지 확인가능
ex) skt_landing.action.loading.on({ta:'셀렉터',co:'white',size:true}); // 설정
ex) skt_landing.action.loading.off({ta:'셀렉터'}); // 해제
ex) skt_landing.action.loading.on({ta:'셀렉터',co:'blue',size:false}); // 설정
ex) skt_landing.action.loading.off({ta:'셀렉터'}); // 해제
ex) skt_landing.action.loading.allOff(); // 전체해제
