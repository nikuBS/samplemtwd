    • 리필쿠폰을 보유 했더라도 데이터/음성 기본 제공량 소진 시 자동으로 사용되지 않으며, 반드시 직접 ‘리필하기’ 를 신청한 경우에만 사용이 가능합니다.
    • 리필한 데이터/음성은 당월에 한해 사용 가능합니다.
    • 일시정지 기간은 가입 기간 산정 시 제외되나, 리필쿠폰 유효기간에는 포함됩니다.
    • 리필하기 신청 이전에 발생한 데이터/음성 초과 사용량에 대해서는 정상 과금 됩니다.
    • 기본제공 데이터 또는 음성이 남은 상태에서도 리필하기 사용이 가능합니다.
    • 리필쿠폰은 SK텔레콤 2년 이상 이용시, 가입일 기준으로 익월 1일에 발급됩니다. (단, 1년 내 정지기 제외/갱신일자 확인필요)
  • 2. 대상 요금상품 조건
    • 팅 계열 요금상품 이용 고객이 음성 리필하기 선택 시, 고객이 설정한 데이터 상한금액을 제외한 기본 제공량의 20%를 추가 제공합니다.
      • 예) LTE팅34 요금상품 이용 시, 데이터 상한금액(5/10/15천원 중 택 1)을 제외한 기본 혜택 금액의 0% 추가 제공
    • 음성 기본 제공량의 유형이 2가지 이상인 요금상품은, 발신 대상이 제한된 음성 제공량(예: 망내, 커플, 수 그룹) 외의 음성 기본 제공량을 추가 제공합니다.
      • 예) T끼리 35 요금상품은 망외 음성 제공량 / LTE 커플 52 요금상품은 기본제공 음성통화량 기준 리필 추가 제공
    • 전국민 무한75, 온가족행복플랜78 이용 시, 유선+영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공 전국민 무한 85/100, 온가족행복플랜85/100 이용 시, 영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공
    • 데이터 무제한형 요금상품의 경우, 음성 리필만 가능합니다. (예 : 올인원 54/64/79/94 등)
    • 온가족행복플랜의 경우, 음성/데이터 추가 혜택 제공량은 리필하기 대상에서 제외됩니다.
