- .widget 다음에는 .widget-box 만 올수있음
- .widget-box 에는 정해진 class명 외에 사용금지 widgets.js 참조
- 특정 style이 필요할시 .widget .widget-box 두 태그의 구조만 바꾸지 않고 자유롭게 수정가능
ex) .c100에 data-percent로 퍼센테이지 지정가능
ex) .c100 > div 에 class 지정가능
ex) loop는 여러개 반복 가능
