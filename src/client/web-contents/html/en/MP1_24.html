Things to Note

Things to Note

  • Free T map access (only for T map compatible phones)
  • LTE data contract discount and tablet contract discount not allowed
  • Internet-based data phone calls (mVoIP) not allowed
  • Cannot subscribe to discount plans such as ‘T We-Family’, ‘TB We-Family Free’, ‘TB We-Family Complimentary’ (not recognized as family line number)
  • ‘Data Sharing among T’, ‘Data Gifts’, ‘Data Refill’ services not allowed

Sharing Settings

  • 5G Tab (Sharing) supplementary lines may be linked to smart phone lines registered under the same name with SK Telecom
  • When cancelling the main line connected with 5G Tab 4G (Sharing) or changing to plans that cannot share data, 5G Tab 4GB (Sharing) supplementary lines that are linked to the main line will be automatically changed to 5G Tab 4GB (Standalone)

Others

  • Tablets supporting talk/text will be charged according to standard rates
    • Talk: KRW 1.98/sec, KRW 3.3/sec, Text: SMS KRW 22/LMS KRW 33/MMS KRW 110 (VAT included)
  • Main Line Plans
    • Only 5G plans (5GX Platinum/Prime/Standard, Slim)
    • Shareable data for each plan: 5GX Platinum - 50GB, 5GX Prime - 30GB, 5GX Standard - 20GB, Slim - data can be shared until the data package is depleted
  • Devices
    • Only 5G tablets
  • Data Consumption Order
    • 5G Tab 4GB (Sharing): Data package of supplementary Line (4GB) → T data coupon data of supplementary line → Refilled data of main line → Data package of main line → Speeds reduced to 1Mbps
    • 5G Tab 4GB (Standalone): Data package of supplementary line (4GB) → T data coupon data of supplementary line → Speeds reduced to 1Mbps
