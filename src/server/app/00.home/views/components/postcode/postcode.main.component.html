우편번호 조회

  • 도로명주소
  • 지번주소
  • 사서함

  • 도로명 주소 조회 (1/3)

    찾으시려는 지역을 선택하시고 도로명을 입력 후, 조회 버튼을 눌러주세요.

    예) 서울시 강남구 선택, “봉은사로” 입력

  • 지번주소 조회 (1/3)

    찾으시려는 주소지의 동/읍/면/리 이름을 입력 후, 조회 버튼을 눌러주세요.

    예) ‘서울 강남구 삼성동’의 경우 ‘삼성’ 입력

  • 사서함 조회 (1/3)

    찾으시려는 주소지의 사서함 이름을 입력 후, 조회 버튼을 눌러주세요.

    예) 서울 영등포구 선택,’여의도 우체국의 경우 ‘여의도’ 입력
