휴대폰 번호 * 해당 휴대폰이 다른 아이디에 등록되어 있을 경우 삭제 후 등록 가능합니다.

닉네임(선택)
  • 닉네임은 다회선을 사용하시는 고객님의 경우 각 회선을 구분할 수 있도록 고객님께서 직접 만들 수 있는 회선명입니다.
  • 닉네임은 7자리까지 입력가능하며 특수문자 및 2자리 이상의 숫자는 입력할 수 없습니다.
