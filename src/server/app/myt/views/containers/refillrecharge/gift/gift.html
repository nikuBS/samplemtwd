  • 조르기

    선물 보낸 내용입니다.
꼭 확인해주세요!
    • 선물 후 기본 잔여 데이터가 500 MB 이상인 경우에만 선물하기가 가능합니다.
      • 예) 선물하기 전 잔여 데이터가 700MB인 경우
        200 MB 선물하기 → 가능 (선물 후 잔여 데이터 500 MB)
        300 MB 선물하기 → 불가 (선물 후 잔여 데이터 400 MB)
      • LTE 안심옵션의 무료 데이터는 선물하기 불가
    • 선물하는 고객이 가입한 요금상품에 따라 해당 월 제공되는 데이터 기본 제공량의 최대 50%까지 선물하기 가능합니다.
      (월 중 가입/변경 시, 일할 기본 제공량 적용)
    • SK텔레콤 고객 간에만 선물 가능합니다.
    • 회당 100 MB~ 1 GB까지 선택하여 월 2회/최대 2GB까지 선물할 수 있습니다.
    • T끼리 온가족할인/TB끼리 온가족무료/TB끼리 온가족프리/온가족플랜/온가족행복플랜 가입 고객의 경우 결합되어 있는 가족에게 최대 4회/4 GB까지 선물 가능합니다.
    • T가족결합(착한가족) 가입 고객의 경우 결합되어 있는 가족에게 횟수와 용량 제한없이 선물 가능합니다.
    • 선물 받기 대상 요금상품에서 대상 외 요금상품으로 변경 시, 선물 받은 데이터는 사용할 수 없습니다.
      (단, 선물 받은 당월 내 대상 요금상품 원복 시, 선물 받은 데이터는 다시 사용 가능합니다.)
    • 해당 월말까지 사용가능하며 이월되지 않습니다.
    • 선물 받은 데이터는 타인에게 다시 선물하거나 타 회선과 공유할 수 없습니다.
      따라서, 기본제공 데이터 잔여량 = 가입요금상품의 기본제공 데이터 잔여량 만 표시됩니다.
    • 선물 받은 데이터는 실시간으로 적용되며, 이미 발생한 데이터 초과량을 소급하여 적용되지 않습니다.
    • 선물한 데이터와 받은 데이터는 데이터 선물 내역 및 요금안내서를 통해 확인 가능합니다.
    • 데이터 선물하기는 만 19세가 되는 생일의 해당 월까지 이용할 수 없으며, 만 19세가 되는 생일이 지난 다음달부터 이용이 가능합니다.
    • 선물하기 완료 후 연속하여 선물하기 시도 시, 이전 선물한 데이터 정보반영으로 약간의 시간이 소요되어 선물하기 이용이 일시적으로 제한되니 이용에 참고 바랍니다.
    • T끼리 데이터 선물하기 서비스는 상업적인 목적으로 이용할 수 없습니다.
    • 데이터 판매 등 금전적 이익을 얻을 목적으로 해당 서비스를 이용할 경우, 서비스 이용이 제한 될 수 있습니다.
      이는 인터넷 카페 등을 통한 거래 시 고객 정보 유출로 인한 범죄 악용, 일방적인 거래 취소 등의 금전적 피해 및 고객 간 분쟁이 발생할 수 있어 서비스 이용에 따른 선의의 고객 피해를 사전에 예방하지 위한 목적입니다.
    • 데이터 선물 후 월 기본제공 데이터 잔여량이 500 MB 이상인 요금상품 가용 고객만 가능합니다.
    • 데이터 자동선물 신청하기/받기는 대상 요금상품에 가입되어 있는 고객만 이용 가능합니다.
      (단, 비대상 요금상품 가입 고객의 경우 대상요금상품으로 변경 시 이용 가능합니다.)
    • 자동선물 신청한 데이터는 매월 1일 발송되며, 발송된 데이터는 취소가 불가합니다.
    • 데이터 자동선물 신청은 회당 100 MB ~ 1 GB까지 가능합니다.
    • 데이터 자동선물 신청은 SMS인증이 정상적으로 이루어져야 자동선물이 가능합니다.
    • 데이터 자동선물 신청 할 때와 데이터 발송(매월 1일)할 때 선물한 고객과 받은 고객에게 SMS가 발송됩니다.
    • 요금상품 하향 변경 시 우선순위(자동선물 설정한 순서)에 따라 선물됩니다.
    • T끼리 데이터 선물하기 서비스는 상업적인 목적으로 이용할 수 없으며, 금전 거래 등 이익을 목적으로 해당 서비스를 이용하는 경우 서비스 이용이 제한될 수 있습니다.
    • 선물하기/선물받기 회선 중 하나 이상이 일시정지/명의변경/요금제변경/해지 시 자동 선물하기도 해지될 수 있으며, 변경 철회 시에도 자동선물기능은 원복이 불가하니 필요 시 재 설정하여야 합니다.
