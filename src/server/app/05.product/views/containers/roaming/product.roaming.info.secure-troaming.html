자동안심 T로밍 음성

  • 전고객 매일 3분 음성무료통화로 해외에서 도난 및 분실, 신변 안전 등의 위험 상황에서도 안심하고 통화 가능
  • 무료통화 3분 포함, 하루 최대 30분까지 최대 10,000원(부가세 포함) 부과
  • 음성 수/발신 1분당 과금 기준에서 1초당 과금 기준으로 변경

자동안심 T로밍 데이터

  • 기존 데이터 종량요금 대비 87.5% 저렴한 파격적인 요금(1패킷당 2.2원에서 0.275원으로 인하 / 1MB = 563원 / 10MB=5,630원 / 부가세 포함)
  • 하루 최대 5,000원(부가세 포함)까지, 데이터를 마음껏 사용
