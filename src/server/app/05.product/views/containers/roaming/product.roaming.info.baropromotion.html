무료 체험 프로모션이란?

baro 통화는 T전화를 이용해 해외에서도 고품질의 데이터 음성통화를 할 수 있는 서비스입니다.
baro 무료 체험 프로모션으로 제공되는 로밍 데이터 10MB(400kbps 이하 속도)로 baro 통화를 체험해보세요.

baro 통화를 프로모션으로 만나보세요.

SK텔레콤 고객님이라면, 데이터 로밍 요금제에 가입하지 않고 무료로 체험하실 수 있습니다.

혜택 제공 횟수

월 1회 제공

  • 로밍 데이터 10MB를 다 쓴 후에는 데이터가 자동으로 차단됩니다.
  • baro 통화를 계속 사용하고 싶다면 데이터 충전 상품을 이용하거나 로밍 데이터를 넉넉하게 사용할 수 있는 데이터 로밍 요금제에 가입해주세요

무료 체험 프로모션 이용 안내

  • Wi-Fi가 없는 곳에서도 baro 무료 체험 프로모션을 이용하시면, 고품질의 데이터 음성통화를 하실 수 있어요.
  • 프로모션을 통해 약 20분 동안 무료로 통화해 보세요.
