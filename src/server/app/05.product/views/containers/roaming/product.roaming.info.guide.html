로밍시 꼭 알아야할 포인트

01

출국 전 준비

  • 1. T 로밍 요금제 가입은 필수!
    • OnePass300 등 내게 맞는 로밍요금제를 가입하세요. 자동 업데이트 APP등으로 인한 과도한 요금 발생을 방지할 수 있습니다.
    • 자세히 보기
  • 2. LTE 로밍 설정
    • LTE 로밍 이용 고객은 로밍 설정 필수!
    • 기간형 요금제를 이용하셨던 고객은 데이터 로밍 차단 상태이므로, 신규 요금제를 가입하시거나 데이터로밍차단 서비스를 해지하셔야 로밍이 가능합니다.
  • 3. 이용가능 지역(커버리지) 확인
    • 미국/멕시코/베트남/브라질/인도/인도네시아/일본/태국/캐나다/홍콩을 방문하시는 고객님께서는 T 로밍 고객센터에서 상세 커버리지(이용 가능 지역) 및 안내를 받으시기 바랍니다.
    • 3G만 탑재된 핸드폰의 경우 T 로밍 고객센터를 통해 상세 도시별 커버리지(이용 가능 지역)를 반드시 확인하시기 바랍니다.
  • 4. 데이터 차단 무료 부가서비스
    • 데이터 로밍 차단 기능 없는 스마트폰의 경우 해외에 나가시기 전 3G 데이터 기능을 차단하시고 나가셔야 합니다.
    • 자세히 보기

02

현지에서

  • 1. 로밍요금제 가입을 잊었어요!
    • 데이터 접속료 무료인 모바일 T 로밍 홈페이지 (m.troaming.co.kr 또는 T 로밍.com)로 접속하시거나 +82-2-6343-9000(해외에서 SKT폰 무료)으로 전화해주세요.
  • 2. 로밍 요금 / 안내문구 확인!
    • 해외 도착 후 국가별 로밍요금, 데이터사용 유의 사항 등 휴대폰으로 발송되는 안내문구를 반드시 확인하세요.
  • 3. 전화연결, 통화품질 안내
    • 15XX, 080 등 번호는 해외에서 연결이 불가능 합니다. 단, 접속이 되었을 경우 국제전화요금이 과금될 수 있으니 주의하세요.
    • 홍콩 2G 자동로밍 고객은 출국 전 국내에서 "15+통화"를 통해 새로운 프로그램을 다운로드 받으시기 바랍니다.
    • 해외 로밍 중 음성 전화 수신시 '수신거절' 버튼 누르는 경우에도 요금이 부과될 수 있으니 주의하시기 바랍니다.
    • 현지 통신망 사정에 따라 원활한 서비스가 제공되지 않을 수 있습니다.
    • 방문하시는 지역의 통신망 사정이나 천재지변 등으로 원활한 서비스가 제공되지 않을 경우, T 로밍 고객센터(해외 +82-2-6343-9000 무료 / 02-6343-9000 무료)로 문의해주시기 바랍니다.
  • 4. 휴대폰 / USIM을 분실했어요!
    • 과도한 로밍요금이 발생할 수 있으므로 고객센터로 즉시 신고하세요! +82-2-6343-9000(SKT폰 무료)

03

귀국 후

  • 1. 데이터 로밍무조건 차단 해지 신청하기
    • 다음 출국 시 T 로밍 서비스를 이용하시려는 고객은 데이터 로밍무조건 차단 서비스를 해지하세요. 해지방법 : T 로밍센터, 고객센터 이용(주말/공휴일 114 고객센터)
    • T 로밍 센터 자세히 보기
  • 2. 한국에 왔는데 3G/LTE 접속이 안됩니다!
    • 첫째, 휴대폰 상태를 확인해 주세요!
      • 1) 안드로이드
        • -설정 > 무선 및 네트워크 > 데이터 네트워크 설정 '허용' 상태 확인
        • -설정 > 무선 및 네트워크 > 모바일 네트워크 > APN > SK Telecom 체크 확인! (SK Telecom Roaming은 사용이 안됨)
      • 2) 아이폰
        • -설정 일반 네트워크 셀룰러 데이터 'ON' 상태인지 확인
    • 둘째, 휴대폰을 껐다 켜 주세요.
    • 셋째, T 로밍 고객센터 및 T 로밍센터로 문의해주세요.

04

기타

  • 1. 요금청구/ 부과안내
    • T 로밍은 별도의 가입비, 기본료가 없습니다.
    • 사용요금 청구는 서비스 사용하신 달의 1개월 후 청구되며, 2~3개월 후 까지 요금이 청구될 수 있습니다.
  • 2. 요금제 안내
    • 로밍서비스 이용시 국내요금제(커플, 지정번호 할인 등)에 따른 할인 등은 적용되지 않습니다.
    • 부가세는 로밍요금 전체 금액의 10%입니다.
    • 로밍요금은 해외사업자 또는 국제전화사업자의 정책이 바뀌는 것에 따라 변경될 수 있습니다.
  • 3. 데이터 로밍 기본정보
    • 데이터 로밍은 전세계 패킷(0.5KB)당 0.275원(부가세포함)으로 단일화되었습니다.
    • T로밍 데이터 안심서비스!
      데이터 종량 과금이 5,000원 일 상한 도달 시, 원패스 제휴 사업자 접속의 경우 200kbps 속도제어, 비 제휴 사업자 접속의 경우 데이터 차단이 됩니다.(단, 2G 단말 이용 고객은 자동안심 T로밍 서비스 이용이 불가합니다.)
    • 전 세계 주요국에서 T로밍 OnePass300 요금제 이용이 가능합니다. 해당 국가 여행 시 데이터 로밍이 필요할 경우 요금제를 가입하여 이용하시면 안심하고 저렴하게 이용이 가능합니다.
    • T로밍 OnePass300 자세히 보기
  • 4. 스마트폰/태블릿 로밍 주의사항
    • 태블릿 PC도 일반 스마트폰과 동일하게 데이터 로밍 서비스가 제공됩니다.
    • 태블릿PC도 안전한 데이터 사용을 위해서는 데이터 요금제 가입이 필요합니다.
    • 데이터 로밍을 이용하지 않으실 경우에는 데이터 로밍 무조건 차단 서비스에 가입하시거나, 휴대폰에서 데이터 차단을 하시기 바랍니다.
      • *데이터 차단 고객의 경우 다음 출국시 휴대폰 내 데이터 차단을 하지 않으셔도 무료로 자동차단됩니다. (데이터 종량 과금이 5,000원 일 상한 도달 시, 원패스 제휴 사업자 접속의 경우 200kbps 속도제어, 비 제휴 사업자 접속의 경우 데이터 차단이 됩니다.)
  • 5. 사용상 기본정보
    • LTE스마트폰도 3G 데이터 로밍 이용이 가능합니다.
    • 태블릿 PC로 3G 데이터 로밍을 이용할 때도 데이터 로밍 월상한 서비스가 적용됩니다.
    • 특정 비행기 및 선박에서는 별도의 로밍 신청 없이도 자동로밍 사용이 가능하며, 국내 요금제와 관계 없이 별도의 로밍 요금이 부과됩니다.
    • 항공 로밍 자세히 보기
