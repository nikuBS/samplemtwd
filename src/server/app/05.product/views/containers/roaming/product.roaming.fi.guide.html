Why baro Box

SK텔레콤 고객님에게만 제공되는 혜택
1 고객님의 휴대폰에 로밍 데이터 500MB 추가 제공
  • 함께 여행하는 일행과 잠시 떨어졌을 때
  • 여행 중 일행과 다른 일정이 있을 때
  • baro box 배터리가 없을 때
2 국내처럼 빠르고 생생한 품질의 baro 통화 이용
  • 해외에서 고품질의 데이터 음성통화
  • 국내 휴대폰, 집 전화 어디로 걸든 자유롭게
  • 로밍 요금제를 이용한 음성통화는 무료(단, 제3국 발신 제외)
3 여행 내내
넉넉한 데이터
  • 1일 데이터 1GB 제공, 1GB 다 쓴 후에도 최대 400kbps 속도로 이용
  • 혼자는 물론 둘이서 사용해도 충분한 데이터
4 여러 명이 함께
이용 가능
  • 1명이 가입한 요금제로 여러 명의 고객님이 이용
  • 여러 명이 함께 여행 간다면 1인 부담 비용 절감

baro Box 대여 요금 안내

baro Box 대여 요금 안내이며 요금제, 이용요금(부가세 포함) 안내입니다.
baro Box 글로벌 기간형 7,000원/일(부가세 포함)
baro Box 일본 3,000원/일(부가세 포함)
baro Box 미국, 캐나다 4,000원/일(부가세 포함)
baro Box 아시아 4,000원/일(부가세 포함)
baro Box 요금제 공통
baro Box 요금제 공통 기본 제공 데이터양과 추가혜택 안내입니다.
기본 제공 데이터양 1GB/일 + 속도 제한
(1GB 소진 후 최대 400kbps 속도로 계속 사용 가능)
로밍 데이터 500MB 추가 제공 혜택
(baro Box 대여 기간 동안 사용 가능)
  • 기본 제공 baro Box 데이터를 다 쓴 후에는 최대 400kbps 속도로 계속 사용하실 수 있습니다.
  • 임대료는 임대기간 동안 적용됩니다.
