설정 방법

휴대폰에서 LTE 로밍 설정

LTE 데이터 로밍을 이용하기 위해 휴대폰에서 LTE 사용 모드 및 지정된 사업자 망을 설정해주세요. 현지 사정에 따라 LTE 데이터 통신이 원활하지 않은 경우 3G 데이터로 접속될 수 있습니다.

iOS: 아이폰 6S(iOS9 기준)
  1. '설정'에서 '셀룰러' 선택
  2. 셀룰러 데이터 ON
  3. 셀룰러 데이터 옵션 > 데이터 로밍 ON
  4. 셀룰러 데이터 옵션 > LTE 활성화
    (음성 및 데이터 / 데이터 전용 중 선택)
안드로이드
  1. 환경 설정에서 'T로밍' 선택
  2. '데이터 로밍 설정', 'LTE 로밍 사용' 체크

LTE 로밍 수동 설정

일반폰

메뉴 > 1.T서비스 > 1.글로벌 로밍 > 1.시스템 선택 > 확인 > 2.수동 선택 > 국가별 할인망 선택

스마트폰(iOS)
  1. '설정'에서 '네트워크 사업자' 선택
  2. 자동 OFF
  3. 제휴 이동통신사망 선택
안드로이드
  1. '설정'에서 'T로밍' 선택
  2. 로밍 이동통신사 선택
  3. 수동 선택
  4. 해외 할인망 선택
  • 대부분의 국가에서 SK텔레콤 지정 이동통신사망으로 자동 설정이지만, 혹시라도 안될 경우에는, 위 내용을 참고하셔서 수동설정해 주세요. 지정 이동통신사망 이외에서 로밍 이용 시 과다한 요금이 발생할 수 있습니다.

3G 자동로밍을 쓰신다면?

  • 3G 방식 국가에서 음성은 물론 영상 통화까지 이용
  • 대부분의 3G 휴대폰으로 이용 가능
  • 3G란 3세대 휴대폰으로 영상통화가 되는 기종입니다.

GSM 자동로밍을 쓰신다면?

  • GSM 자동로밍할 수 있는 휴대폰 출시!
  • 유럽 및 일부 동남아 지역에서는 GSM 방식 사용
