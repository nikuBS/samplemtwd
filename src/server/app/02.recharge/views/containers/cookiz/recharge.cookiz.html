    • 충전 금액 내에서만 당월 사용이 가능하며, 사용한 금액에 대해서는 익월 청구서에 청구됩니다.
    • 충전 금액을 사용하지 않았을 경우 당일에 한해서만 취소 가능합니다. 이후에는 충전 금액을 사용하지 않은 경우라도 취소가 불가합니다.
    • 충전 선물한 금액은 부가세를 포함하지 않은 금액이며, 부가세는 선물 받으신 분의 요금에 포함되어 청구됩니다.
    • 상한금액은 음성통화, 폰메일(무료제공건수 초과 이용금액 해당), 데이터 통화료에 적용되며 정보료 등 일부 서비스는 적용되지 않습니다.
    • 충전신청 이후부터 바로 음성통화 및 폰메일 이용이 모두 가능합니다.
    • 해당월 내 사용 가능한 상한금액이 초과되면 발신이 정지되고 수신만 가능하며, 발신정지 후 마이벨 기본제공 서비스 이용이 불가합니다.
    • 온가족행복플랜 팅 계열은 온가족행복플랜 결합 여부에 따라 사용중인 요금상품이 정상 조회되지 않을 수 있습니다.
    • 온가족행복플랜 결합을 가입하지 않으신 경우, 추가 혜택이 제공되지 않으며 T world, 고객센터, 지점/대리점을 통해 일반 팅요금상품으로 변경하시기 바랍니다.
    • 월 중 요금상품 변경 또는 일시정지 시 요금상품 기본제공 혜택은 일할 계산되며 기본제공 혜택 초과 이용 시 상한 초과 별도 요금이 청구됩니다.
    • ‘매월 자동충전’은 신청 다음 달부터 신청한 충전금액에 대해 매월 자동으로 충전되어 사용 가능하며 ‘당월 충전’은 신청 당월만 사용 가능합니다.
    • ‘매월 자동충전’은 최종 신청한 금액을 기준으로 매월 자동충전 됩니다.
    • 2009년 09월 28일부터는 법정대리인이 충전허용금액을 사전 설정 시, 충전허용금액 내 충전은 법정대리인 동의 없이 신청 가능합니다.
    • 충전 허용금액 변경은 법정대리인 동의 시 가능하며, 충전 허용금액은 월 최대 2만원 내 설정 가능합니다.
