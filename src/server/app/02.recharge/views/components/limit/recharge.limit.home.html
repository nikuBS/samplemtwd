데이터 한도 요금 충전

내용

요금 한도보다 데이터가 더 필요하다면 충전하여 사용해보세요!

  • 나의 요금제
    <%= limitData.prodName %>
  • 충전 서비스 차단 여부
    <%= limitData.blockYn === "Y" ? "충전 불가" : "충전 허용" %>
데이터 요금 충전
  • 충전 가능 금액 <%= limitData.currentTopUpLimit %> 원
데이터 차단 설정
  • 이번 달 해제 ON/OFF스위치 차단
  • 다음 달부터 해제 ON/OFF스위치 차단
  • 충전 금액은 0.5KB 당 0.025원을 적용하여 실제 사용한 부분에 대해서만 후불 청구됩니다.
  • 충전은 매월 5,000원 단위로 최대 20,000원까지 가능합니다.
  • 충전 서비스 차단 시 요금 충전이 불가능 합니다.
  • 데이터 한도 요금 충전 내역

꼭 확인해주세요!

  • 1. 충전 금액
    • 0.5KB 당 0.025원을 적용하여 실제 사용한 부분에 대해서만 후불 청구됩니다.
    • 충전 후 데이터를 1패킷 (0.5KB) 이상 사용하실 경우 충전 금액이 적용됩니다.
    • 충전은 매월 5천원 단위로 최대 2만원까지 가능합니다.
  • 2. 충전 데이터
    • 충전한 데이터는 충전 당월에 한하여 사용이 가능합니다.
  • 3. 매월 자동 충전
    • '매월 자동충전'은 신청 다음 달부터 신청한 충전금액에 대해 매월 자동으로 충전되어 사용 가능하며 '당월 충전'은 신청 당월만 사용 가능합니다.
    • ‘매월 자동충전’은 최종 신청한 금액을 기준으로 매월 자동충전 됩니다.
  • 4. 차단 해제
    • 차단 해제는 무선인터넷 차단을 해제하여 데이터 이용이 계속 가능하도록 하는 기능입니다.
    • 데이터 차단 해제 신청은 월 3회까지 신청 가능하며, 신청 취소는 당월 중 가능합니다.
  • 5. 요금 상품
    • 팅요금상품은 팅요금상품 충전 한도내에서만 충전 가능하며, 별도 데이터한도 충전이 불가합니다.
    • 기본제공 통화는 국내 데이터 통화에 한하며, 콘텐츠이용료, 영상통화, 컬러메일 건당 요금 등은 별도 과금 됩니다.
