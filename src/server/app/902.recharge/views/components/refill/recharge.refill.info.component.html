꼭 확인해주세요!

    • 리필쿠폰을 보유했더라도 데이터/음성 기본 제공량 소진 시 자동으로 사용되지 않으며, 반드시 직접 ‘리필하기’ 를 신청한 경우에만 사용이 가능합니다.
    • 리필한 데이터/음성은 당월에 한해 사용 가능합니다.
    • 일시정지 기간은 가입 기간 산정 시 제외되나, 리필쿠폰 유효기간에는 포함됩니다.
    • 리필하기 신청 이전에 발생한 데이터/음성 초과 사용량에 대해서는 정상 과금 됩니다.
    • 기본제공 데이터 또는 음성이 남은 상태에서도 리필하기 사용이 가능합니다.
    • 리필쿠폰은 SK텔레콤 2년 이상 이용시, 가입일 기준으로 익월 1일에 발급됩니다. (단, 1년 내 정지기 제외/갱신일자 확인필요)
    • 팅 계열 요금상품 이용 고객이 음성 리필하기 선택 시, 고객이 설정한 데이터 상한금액을 제외한 기본 제공량의 20%를 추가 제공합니다.
      • 예) LTE팅34 요금상품 이용 시, 데이터 상한금액(5/10/15천원 중 택 1)을 제외한 기본 혜택 금액의 0% 추가 제공
    • 음성 기본 제공량의 유형이 2가지 이상인 요금상품은, 발신 대상이 제한된 음성 제공량(예: 망내, 커플, 수 그룹) 외의 음성 기본 제공량을 추가 제공합니다.
      • 예) T끼리 35 요금상품은 망외 음성 제공량 / LTE 커플 52 요금상품은 기본제공 음성통화량 기준 리필 추가 제공
    • 전국민 무한75, 온가족행복플랜78 이용 시, 유선+영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공 전국민 무한 85/100, 온가족행복플랜85/100 이용 시, 영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공
    • 데이터 무제한형 요금상품의 경우, 음성 리필만 가능합니다. (예 : 올인원 54/64/79/94 등)
    • 온가족행복플랜의 경우, 음성/데이터 추가 혜택 제공량은 리필하기 대상에서 제외됩니다.
    • 유효기간이 가장 짧은 쿠폰을 먼저 사용해주시기 바랍니다.
    • 쿠폰은 반드시 유효기간 내에 모두 사용하셔야 합니다.
      • ※ 유효기간 내에 사용하지 않은 쿠폰은 자동 소멸됩니다. (단, 리필쿠폰의 유효기간이 남아 있는 경우라도 다음 가입년수 갱신시점에 자동 소멸됩니다.)
      • 예) 리필하기 최초 출시시점(2013년 4~5월)에 발급된 리필쿠폰 또는 가족간 명의변경이나 번호이동 철회 후 재가입 등으로 장기가입기준일자 변경된 경우 자동소멸
    • 리필하기 이용 시, 사용중인 요금제의 월 기본 제공 데이터의 100%만큼 추가 사용 가능합니다.
      • 예) LTE62 요금제 이용 고객이 리필하기 시, LTE 62 요금제의 기본 제공 데이터인 5GB 추가 제공
    • 음성 기본 제공량이 있는 경우, 데이터 리필하기 대신 음성 리필하기로 선택하여 사용 가능합니다. 단, 음성 리필하기 선택 시 기본 제공 음성 통화량의 20%만 리필 가능
      • 예) LTE62 요금제 이용 고객이 리필하기 시, LTE 62 요금제의 기본 제공 음성 350분의 20%인 70분 추가 제공
    • 팅 계열 요금제 이용 고객이 음성 리필하기 선택 시, 고객이 설정한 데이터 사용가능 금액을 제외한 혜택 금액의 20%를 추가 제공합니다.
      • 예) LTE팅34 요금제 이용 시, 데이터 상한금액 (5/10/15천원 중 택 1)을 제외한 기본 혜택 금액의 20% 추가 제공
    • 음성 기본 제공량의 유형이 2가지 이상인 요금제는, 발신 대상이 제한된 음성 제공량(예: 망내, 커플, 특수 그룹) 외의 음성 기본 제공량을 추가 제공합니다.
      • 예) T끼리 35 요금제는 망외 음성 제공량 / LTE 커플 52 요금제는 기본제공 음성통화량 기준 리필 추가 제공
    • 전국민 무한75, 온가족행복플랜78 이용 시, 유선+영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공 전국민 무한 85/100, 온가족행복플랜85/100 이용 시, 영상+부가통화 300분 무료제공량의 음성 리필 20% 추가 제공
    • 기본 제공량 소진순서는 다음과 같습니다.
      • * 데이터 리필 : 리필한 데이터 > 프로모션 데이터 > 선물 받은 데이터 > 기본 제공량
      • * 음성 리필 : 리필한 음성 > 기본 제공량
    • 리필 쿠폰선물 후 취소가 불가하며, 선물 이전의 유효기간과 동일하게 적용됩니다.
    • 월 1회, 연간 최대 2회까지 선물하기가 가능합니다.
    • 가족간 리필쿠폰 선물하기는 대상 결합상품 가입고객만 이용 가능합니다.
      • ※ 대상 결합상품 : 온가족할인/T가족결합(착한가족)/온가족무료/온가족프리/온가족행복플랜
      • ※ 온가족행복플랜 : 음성/데이터 추가 혜택 제공량 리필하기 대상 제외
    • 팅 계열 요금상품을 이용중인 고객은 가족간 리필쿠폰 선물 받기만 가능하며, 선물하기는 불가합니다.
    • 선물 받는 고객이 가입기간 2년 이상인 경우, 선물 받는 고객의 장기가입일자를 기준으로 연 2회 선물 받기가 가능합니다.
      • ※ 단, 가입기간 2년 미만 고객은 서비스 개통시점부터 2년차 쿠폰 발급 전까지 총 2회 선물 받기가 가능합니다.
    • 리필하기로 데이터 또는 음성이 제공된 이후에는 추가 제공량의 이용 여부와 무관하게 이용취소가 불가합니다.
    • 리필하기 신청 후 요금제를 변경하면 리필한 데이터/음성통화량은 이용할 수 없으나, 리필하기 신청한 시점의 요금제로 당월 원상 복귀하면 잔여 리필 데이터/음성통화량을 사용하실 수 있습니다.
    • 리필한 데이터는 ‘T끼리 데이터 선물하기’ 서비스 이용이 불가합니다.
