로그인 실패 횟수 / 하루 로그인 시도 횟수 제한제도를 안내해드립니다.

로그인 실패 횟수 제한제도란?

지속적인 로그인 시도로 개인정보가 유출되는 것을 방지하기 위한 제도입니다. T world에서 고객님의 아이디로 연속 5회 이상 로그인 실패할 경우, 로그인이 제한됩니다.

  • 로그인 5회 실패할 경우 초기화 방법

    본인인증 후 비밀번호를 재설정합니다.

    연속으로 로그인 5회 실패 후, 정상 비밀번호를 입력해도 로그인하실 수 없습니다.

    이 경우 T아이디 비밀번호 재설정 페이지에서 본인인증 후, 비밀번호를 다시 설정해야 로그인하실 수 있습니다.

    로그인 실패 횟수는 온라인 T world와 T world 앱에서 합산되며, 임의로 초기화되지 않습니다.

    자동로그인 및 기타 앱의 사용 여부를 확인해 주세요.

    로그인 실패 횟수 합산 방식 안내

    T world와 같은 계정을 사용하는 T world(온라인 / 모바일),
    T멤버십(온라인 / 모바일), T world 다이렉트, T로밍 서비스에서의 로그인 실패 횟수를 합산합니다.

하루 로그인 시도 횟수 제한제도란?

T world는 고객님의 개인정보보호를 위해, 하루 로그인 시도 횟수를 30회로 제한합니다.
30회 이상 로그인하신 경우, 다음날 00시 이후부터 정상적으로 로그인하실 수 있습니다.

T world 앱 이용안내

단, T world 앱은 하루 로그인 시도 횟수에 제한이 없습니다.

  • 하루 로그인 시도 횟수를 초과하지 않았는데
    접속이 제한된 경우

    T world와 같은 계정을 사용하는 다른 서비스의 당일 로그인 여부를 확인합니다.

    하루 로그인 시도 횟수는 T world와 같은 계정을 사용하는 모든 서비스에서 합산됩니다.

    T world(온라인 / 모바일), T멤버십(온라인 / 모바일), T world 다이렉트, T로밍 서비스에서 당일 여러 번 로그인을 시도하셨는지 확인해 주세요.

    ‘자동로그인 플러스’ 기능 사용 여부를 확인합니다.

    만약 ‘자동로그인 플러스’ 기능을 설정하셨다면 일정 시간 간격을 두고 지속적으로 로그인이 시도되었을 수 있습니다.
    이 경우, ‘자동로그인 플러스’ 해제 후 이용해 주세요.

    자동로그인 플러스란?

    SK텔레콤의 자동로그인 기능입니다. 해당 기능을 설정하면 SK텔레콤에서 제공하는 모든 앱에서 아이디와 비밀번호 입력 없이 간편하게 로그인하실 수 있습니다.
