로그인 실패 횟수 / 하루 로그인 시도 횟수 제한제도를 안내해 드립니다.

로그인 실패 횟수 제한 제도란?

지속적인 로그인 시도로 인한 개인정보 유출을 방지하기 위해 T 월드에서는 고객님의 아이디로 연속 5회 이상 로그인 실패할 경우, 로그인을 제한하고 있습니다.

  • 로그인 5회 실패할 경우, 초기화 방법

    본인인증 후 비밀번호 재설정을 합니다.

    연속으로 로그인 5회 실패 후, 정상 비밀번호를 입력해도 로그인은 할 수 없습니다.
    이 경우, T 아이디 비밀번호 재설정 페이지에서 본인인증 후 비밀번호 재설정을 진행하셔야 로그인이 가능합니다.

    로그인 실패 횟수는 온라인 T월드와 모바일 T월드 앱에서 누적되며 임의로 초기화되지 않습니다.

    자동 로그인 및 기타 앱의 사용 여부를 확인해 주세요.

    로그인 실패 횟수 합산 방식 안내

    T월드와 같은 계정을 사용하는 T월드(온라인, 모바일),
    T멤버십(온라인, 모바일), T월드 다이렉트, T로밍
    로그인 실패 횟수를 합산합니다.

1일 로그인 시도횟수 제한 제도란?

T월드는 고객님의 개인정보 보호를 위해 1일 로그인 시도 횟수를 30회로 제한합니다.
30회 이상 로그인하신 경우, 다음날 00시 이후부터 정상적으로 로그인하실 수 있습니다.

모바일 T월드 앱 이용방법 안내

단 모바일 T월드 앱은 로그인 횟수 제한이 없습니다.

  • 1일 로그인 제한 횟수를 넘지 않았는데 접속이
    제한된 경우

    T월드와 같은 계정을 사용하는 다른 T서비스가 같은 날 로그인 실패 했는지 확인합니다.

    1일 로그인 시도 횟수는 T월드와 같은 계정을 사용하는 모든 T서비스를 합산합니다.
    T월드(온라인, 모바일), T멤버십(온라인, 모바일), T월드 다이렉트, T로밍 서비스에서 당일 여러 번 로그인을 시도하셨는지 확인해 주세요.

    ‘자동로그인 플러스’ 기능 사용했는지 확인합니다.

    만약 ‘자동로그인 플러스’ 기능을 설정하셨다면 일정 시간 간격을 두고 지속적으로 로그인이 시도되었을 수 있습니다.
    이 경우, ‘자동로그인 플러스’ 해제 후 사용해 주세요.

    자동로그인 플러스란?

    SKT의 자동 로그인 기능이며, 기능 설정 시 SKT에서 제공하는 모든 앱에서 아이디와 비밀번호 입력하지 않아도 간편하게 로그인하실 수 있습니다.
