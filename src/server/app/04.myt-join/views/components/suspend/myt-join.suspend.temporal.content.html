일시정지 기간

title="시작날짜 선택" disabled="" data-role="fe-date-from"/> ~ title="마지막날짜 선택" data-role="fe-date-to"/>
  • 일시정지는 1회 최대 3개월까지 신청 가능하며 3개월 경과 시 자동 해제됩니다.
  • 일시정지는 월 4회 정지기간은 연 최대 180일 초과가 불가합니다.
  • 일시정지 기간 익일에 자동으로 일시정지 해제 됩니다.

일시정지 설정

  • 걸기/받기 모두 정지

  • 걸기만 정지

일시정지 종료 7일 전

  • 받기 가능 상태로 자동 전환

  • 받기 불가 상태로 유지

  • 단, 받기 가능한 상태로 전환된 후 이용 가능한 유료 부가서비스 예로 착신전환, 수신자부담 서비스 등은 정상과금됩니다.
  • 받기 불가 상태로 유지시 정지 종료 SMS는 안내하지 않습니다.

일시정지 종료일

  • 이메일 또는 문자로 안내

  • 별도 안내 없이 자동해제

  • 이동전화번호는 SK회선일 경우에만 발송이 가능합니다.
  • 이메일로 안내
    이메일 주소
  • 문자로 안내
