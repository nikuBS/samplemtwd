    서비스 해지 접수신청 시 아래사항을 꼭 확인 후 신청해주시기 바랍니다.

    1. 해지 접수 시 당사에서 임대한 장비는 회수 되며, 만약 장비를 분실하시거나 장비가 손된 경우는 변상금이 청구 될 수 있습니다.
    2. 인터넷 해지신청 시 인터넷전화의 사용이 제한될 수 있습니다.(자세한 상담을 원하시면 전화 1600-2000(유료) 또는 080-816-2000(무료)로 확인 바랍니다.)
    3. 세트 할인을 받고 있는 고객의 경우 인터넷을 해지하시게 되면 사용중인 인터넷전화 또는 B tv 월정액이 인상됩니다.
    4. 해지 시 고객님의 기존 프로모션 혜택은 소멸되고, 약정만료 전에 해지 시 할인 반환금 및 사은품 위약금이 발생합니다.
    5. T 인터넷 + B tv 상품의 경우 해지 취소 신청 시 B tv도 자동으로 해지 취소 됩니다.
    6. 초고속 인터넷의 경우 해지 후 동일 주소, 동일 고객 명의로 3개월 이내 신규 가입할 수 없습니다.
    7. T월드를 통한 해지 신청의 경우, 신청하신 해지 요청 일에 자동 해지 됩니다.
    8. 인터넷 전화장비 위약금 조회는 관련 상세내역 조회 시 전화 1600-200(유료) 또는 080-816-2000(무료)로 문의하여 주시기 바랍니다.
