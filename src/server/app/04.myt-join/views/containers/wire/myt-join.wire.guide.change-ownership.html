명의 변경 신청 방법 안내

내용

  1. 01전화로 신청

    • 1600-2000(유료)
      또는 080-816-2000(무료)을 통해서 신청

  2. 02서류 준비 후 팩스로 발송

    • 이용권승계신청서, 양수/양도인 or 사업자등록증

  3. 03상담사 통화

    • 상담사가 양수/양도인 통화 후 변경 처리
  • 이용권승계 신청서, 양수/양도인 신분증 앞면 또는 사업자 등록증 (신분증의 경우 주민등록번호 뒷자리는 가려주세요)
  • 양수인의 주소와 유선서비스 설치 장소의 주소가 동일하지 않을 경우에는 설치장소 동일함을 확인하는 서류 필요
  • 이용권 승계 신청서는 상담사를 통해 팩스로 수신 or 해당 페이지 상단의 [이용권 승계 신청서 다운로드] 이용
  • 단, 증빙서류는 최근 3개월 이내에 발급 받은 것이어야 함

꼭 확인해 주세요
