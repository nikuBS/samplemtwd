이용안내

    • 휴대폰 일시정지 시, T 데이터셰어링 서비스를 이용하실 수 없습니다.
    • T 데이터셰어링을 해지한 USIM으로는 데이터 서비스를 사용하실 수 없습니다. 사용을 원하시는 경우, 다시 서비스 등록을 하셔야 합니다.
    • 올인원 등 기본제공 데이터가 있는 요금상품을 이용하시는 고객님께서는 기본제공 데이터 한도를 다 쓰면 서비스를 이용하실 수 없습니다.
    • T 데이터셰어링 서비스 이용 시 시스템에 영향을 줄 경우 데이터 서비스 속도가 느려지거나 이용이 제한 될 수 있습니다.
    • 휴대폰 번호의 명의변경이나 해지 시 등록된 모든 T 데이터셰어링 서비스도 자동 해지됩니다.
    • T 데이터셰어링을 사용할 수 있는 요금상품 (올인원 등 기본제공 데이터 혜택이 제공되는 요금상품 및 안심데이터 등 데이터 옵션 요금상품) 변경 시 등록된 T 데이터셰어링 서비스도 자동 해지됩니다.
    • 올인원 등 요금상품의 경우, USIM 등록일 기준으로 T 데이터셰어링 이용한도 내에서 이용이 가능합니다.
    • USIM 등록일이 2011년 3월 8일 이전 사용자의 경우,
      • - 올인원 34/44 데이터 요금상품 등 : 해당 요금상품의 데이터 한도 내 이용 가능
      • - 올인원 54, 64, 79, 94 계열 : 휴대폰 이용기준과 동일 (데이터 무제한 이용 가능)
    • USIM 등록일이 2011년 3월 8일 이전 사용자의 경우,
      • - 올인원 34/44 데이터 요금상품 등 : 해당 요금상품의 데이터 한도 내 이용 가능
      • - 올인원 54계열 : 700MB
      • - 올인원 64계열 : 1GB
      • - 올인원 79계열 : 1.5GB
      • - 올인원 94계열 넘버원(무제한형) : 2GB.
    • 2011년 3월 8일 이전 등록 USIM 이라도 휴대폰의 요금상품 변경에 따라 데이터 이용 기준이 변경 될 수 있습니다.
      • -2011년 3월 8일 기준 올인원34/44 및 데이터요금제 이용 고객은 2011년 3월 9일 이후 데이터 무제한요금상품으로 변경 하더라도 데이터 이용한도 내에서 이용 할 수 있습니다.
      • - 데이터무제한 요금상품에서 올인원34/44 등 데이터 한도관리가 되는 요금상품으로 변경한 이후에는 데이터 이용 한도 내에서 이용 할 수 있습니다
    • T 데이터셰어링 무제한형 USIM을 사용하는 고객님의 '인터넷직접접속' 사용량은 이번 달 사용량 및 총 데이터 사용량 정보에 합산되지 않습니다.
    • 인터넷직접접속이란 단말기로 URL을 통해 인터넷서비스를 이용하는 것을 말합니다.
