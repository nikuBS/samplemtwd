    • 데이터는 대표회선에서만 나눠쓰기가 가능합니다.
    • T가족결합은 결합회선에 따라 제공되는 데이터량이 달라집니다.
      (이동전화 2회선~3회선: 500MB 제공 / 4회선~5회선: 1000MB 제공)
    • 대표 회선에 총 데이터량이 모두 지급됩니다. 단, 해당 데이터는 나눠쓰기 후(본인포함) 이용할 수 있습니다.

    • 가족끼리 배분한 후 월말까지 미사용한 데이터 또는 배분하지 않은 데이터는 소멸되며, 익월초 신규 데이터가 제공 됩니다. (이월 불가)
    • 대표회선이 가족에게 배분한 데이터는 선물(나눠쓰기) 불가합니다.
