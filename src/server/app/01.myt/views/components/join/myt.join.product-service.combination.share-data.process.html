T가족결합 데이터 나눠쓰기

내용

  • 나눠쓰기 가능 데이터 1,000 MB

받는 분 선택

데이터 나눠쓰기 가족목록

  • 최*영
    010-12**-45** 500 MB 받음
  • 최*석(자녀)
    010-12**-45** 100 MB 받음
  • 박*민(배우자)
    010-12**-45** 받은 적 없음
  • 최*민(형제자매)
    010-12**-45** 받은 적 없음

이용안내

    • 데이터는 대표회선에서만 나눠쓰기가 가능합니다.
    • T가족결합은 결합회선에 따라 제공되는 데이터량이 달라집니다.
      (이동전화 2회선~3회선: 500MB 제공 / 4회선~5회선: 1000MB 제공)
    • 대표 회선에 총 데이터량이 모두 지급됩니다. 단, 해당 데이터는 나눠쓰기 후(본인포함) 이용할 수 있습니다.
    • 데이터는 100MB 단위로 나누어 가족들에게(본인포함) 나눠쓰기 가능하며, 나눠쓰기 횟수는 제한 없습니다.
    • 가족끼리 배분한 후 월말까지 미사용한 데이터 또는 배분하지 않은 데이터는 소멸되며, 익월초 신규 데이터가 제공 됩니다. (이월 불가)
    • 대표회선이 가족에게 배분한 데이터는 선물(나눠쓰기) 불가합니다.
    • 나눠쓰기 한 데이터는 리필한 데이터와 선물받은 데이터를 모두 소진 후 데이터 소진되며, 나눠쓰기 한 데이터 소진 후 기본제공량이 소진됩니다.
    • 나눠쓰기 받는 회선은 데이터 기본제공량이 있는 회선만 가능합니다. 단, 가족등록은 기본제공량이 없는 회선(표준요금제 등)도 가능합니다.
    • 기본제공량이 없는 회선도 대표회선 가능하나, 이 경우 자신에게 나눠쓰기 불가 합니다.
    • 배분한 데이터는 함께쓰기, 선물하기, 테더링, m-VoIP 이용 불가 합니다.
