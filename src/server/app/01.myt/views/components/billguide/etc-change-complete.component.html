이용안내
  • 1. 기타(우편) 요금안내서
    • 기타(우편) 요금안내서 신청 시 기존에 이용하시던 문자/스마트/이메일 요금안내서는 자동 해지됩니다.
    • 기타(우편) 요금안내서 유형 해지 시 기타(우편) 발송 주기도 자동 해지되며, 기타(우편) 청구 유형을 재 선택하면 자동 원복됩니다.
  • 2. 요금안내서 발송
    • 매월 5일까지 신청 시 당월부터 변경된 요금안내서가 발송 되며, 6일 이후 신청 시에는 다음달부터 변경된 요금안내서를 발송해 드립니다.
  • 3. 다회선 청구
    • 여러 개의 회선에 대해 통합요금안내서를 받으시는 경우 통합청구 대표번호로만 신청이 가능하며 전체 금액이 합산되어 발송됩니다.
  • 4. 주소 부정확 사유
    • 주소 부정확 등의 사유로 기타(우편) 요금안내서가 발송되지 못하는 경우 전자요금안내서(Tworld 확인/Bill Letter/문자 요금안내서/이메일 요금안내서)로 변경될 수 있습니다.
