이용안내
  • 1. 포인트 요금 납부 이용안내
    • 본인명의로 이동전화를 등록한 회원에 한해 포인트로 휴대폰금액 납부가 가능합니다.
    • 멤버십 가입 시 OK캐쉬백 카드서비스를 신청하지 않은 고객(OK캐쉬백 주민등록번호 제공 미동의 고객)은 멤버십 카드 번호로 OK캐쉬백 수납이 불가능하오니, 소지하고 계신 타 OK캐쉬백 카드 번호를 입력해주시기 바랍니다.
    • OK캐쉬백 카드가 없으신 고객은 수납이 불가능합니다.
  • 2. 레인보우 포인트
    • 레인보우포인트로 1점당 1원 상당의 음성통화료를 납부하실 수 있습니다. 예) 레인보우포인트 10,000점 차감 시 10,000원 납부
    • 기본료 등에 의해 제공되는 기본제공 통화는 기본적으로 제공 받으실 수 있는 혜택이며, 과금대상이 아니므로 납부대상에서는 제외됩니다.
    • 납부예약 포인트는 최소 1점 이상 부터 입력 가능합니다.
  • 3. OK캐쉬백/T포인트
    • OK캐쉬백 포인트 자동 요금납부를 신청하시면, 매월 청구요금에서 OK캐쉬백 포인트가 자동으로 차감됩니다.
    • 자동납부 시 포인트 보유 잔액과 관계없이 500점 단위로 최대 10,000점까지 신청가능 합니다.
      예) 매월 3천점 자동 요금납부 신청, 7천점 포인트 보유 시에는 잔여 포인트가 4천점이므로 자동차감 되지 않음
    • OK 캐쉬백 포인트 납부 후 잔여미납금은 신용카드/계좌이체/입금전용계좌로 납부하실 수 있습니다. (단, 입금전용계좌는 신청한 고객에 한해 고객님께 부여된 계좌로 납부가 가능합니다.)
  • 4. 미납금액
    • 이용정지 고객의 경우 납부기한이 경과한 미납 금액을 모두 납부하시면 이용정지가 즉시 해제됩니다.
    • 대표회선인 경우에만 미납내역이 조회되며, 대표회선이 아닌 경우 대표회선번호로 로그인 해주십시오
