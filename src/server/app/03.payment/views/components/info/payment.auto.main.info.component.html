꼭 확인해 주세요!
  • 1. 자동납부 신청 안내
    • 최초 인출(승인)예정일은 자동납부 신청 후 확인 하실 수 있습니다.
    • 이용정지 고객의 경우 납부기한이 경과한 미납액을 모두 납부하시면 이용정지가 즉시 해제됩니다
    • 자동납부 할인 적용 기준은 자동납부 인출일 기준으로 다음달에 반영됩니다.예) 2월 청구분을 2월 26일 자동납부로 납부한 경우 : 3월 청구분에서 자동납부 할인 적용2월 청구분을 3월 3일 자동납부로 납부한 경우 : 4월 청구분에서 자동납부 할인 적용
  • 2. 자동납부 해지 안내
    • 은행 자동납부를 해지하시면 자동납부 할인 기준 변경으로 다시 은행자동납부를 신청하더라도 1%할인은 제공되지 않으니 은행 자동납부를 유지하시기를 권장합니다.
    • 매월 5일 이후 자동납부를 해지하시는 경우 다음달에 반영되므로 이번달에는 자동납부로 인출될 수 있으니 참고하시기 바랍니다.
    • 자동납부 해지 시점 및 고객님의 납부방법에 따라 일부 차이가 있을 수 있으니 추가 문의는 당사 고객센터<국번없이 1599-0011(유료) / 휴대폰 114>를 이용해 주시기 바랍니다.
  • 3. 이용안내
    • 모바일 T World에서는 고객님의 휴대폰 명의와 동일한 명의의 은행계좌나 카드로만 신청/변경 하실 수 있습니다.
    • 정기 청구작업으로 인해 매월 5일 21시부터 6일 04시까지 자동납부 신청/변경/해지가 불가합니다.
  • 4. 금융기관의 비 영업일(토/일/공휴일)인 경우 인출(승인)일이 변경될 수 있습니다.
    • 인출(승인)일이 금융기관의 비 영업일(토/일/공휴일)인 경우 다음 영업일로 날짜가 변경될 수 있습니다. 상세한 일정은 매월 요금안내서에 표기 해 드리는 납부일을 참고바랍니다.
